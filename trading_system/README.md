# 자동 매매 시스템 (Auto Trading System)

트레이더 마크 📊 가 구축하는 알고리즘 트레이딩 시스템

## 시스템 구성

### 1. 데이터 수집 모듈
- 실시간 주가 데이터 수집 (Yahoo Finance, KRX, 등)
- 기술적 지표 계산 (이동평균, RSI, MACD, 볼린저밴드 등)
- 기본적 분석 데이터 (재무제표, 뉴스 감성 분석)

### 2. 분석 엔진
- 머신러닝 기반 예측 모델
- 기술적 분석 기반 트레이딩 전략
- 리스크 관리 알고리즘

### 3. 실행 모듈
- 거래소 API 연동 (업비트, 빗썸, 해외 거래소 등)
- 주문 실행 및 포지션 관리
- 실시간 모니터링

### 4. 모니터링 대시보드
- 실시간 포지션 현황
- 수익률 추적
- 시스템 상태 모니터링

## 설치 및 실행

```bash
# 필수 패키지 설치
pip install -r requirements.txt

# 데이터 수집 테스트
python data_collector.py

# 트레이딩 전략 테스트
python backtest.py

# 실시간 트레이딩 실행
python main.py
```

| 파라미터 | 이전 | 이후 | 근거 |

## AI 신호 엔진 개선

### 1. 신호 임계값 조정

**AgentBase 점수 임계값**:
- BUY: 0.25 → **0.15** (더 많은 진입 기회)
- SELL: -0.20 → **-0.15** (대칭성 확보)

**근거**: Freqtrade 평균 임계값 참고

### 2. 최소 신뢰도 기준

**전략별 임계값**:
- AGGRESSIVE: 70% → **60%**
- MODERATE: 75% → **65%**
- CONSERVATIVE: 85% → **72%**

**근거**: Freqtrade/Jesse 베스트 프랙티스

### 3. 변동성 기반 조정

**신뢰도 조정 계수**:
- 최소값: 0.7 → **0.80** (낮은 변동성에서도 거래 가능)
- 변동성 가중치: 3 → **2** (과도한 페널티 완화)

**근거**: 0.1-0.3% 변동성에서도 안정적 거래 가능

## 베스트 프랙티스 근거

### Freqtrade 참고
```yaml
# Freqtrade 일반적인 설정
minimal_roi:
  "60": 0.01  # 60분 후 +1%
  "30": 0.02  # 30분 후 +2%
  "20": 0.03  # 20분 후 +3%
  "0":  0.04  # 즉시 +4%

stoploss: -0.10  # -10% 손절

# 신호 신뢰도
confidence_threshold: 0.60-0.70
```

```python
# Jesse 리스크 관리

minimum_win_rate = 0.40  # 40% 승률 이상
```

```python
# QuantConnect 포지션 관리
max_position_size = account_value * 0.33  # 33% 상한
min_profit_ratio = fee * 2  # 수수료의 2배
```

## 기대 효과

### 단기 (1-2일)
1. **거래 빈도 증가**
   - 신뢰도 완화로 더 많은 진입 기회
   - 현재: 거의 없음 → 목표: 하루 3-5회

2. **수익 품질 향상**
   - 최소 순이익 100원 → 수수료 대비 충분한 마진
   - 미세 손실 (-7원, -15원) 제거

3. **안정성 확보**
   - 최소 보유 10분 → 추세 확인 충분
   - 과도한 회전 방지

### 중기 (1주일)
1. **승률 개선**
   - 목표: 50% 이상
   - 현재 문제였던 손실 패턴 제거

2. **수익률 개선**
   - 현재: -0.5% (49,700원 → 5만원 대비)
   - 목표: +2-5%

3. **시스템 안정성**
   - 예측 가능한 거래 패턴
   - 리스크 관리 강화
